## Overview

MIST-VLA provides tools for improving the safety of Vision-Language-Action models through:

1. **Per-dimension Risk Prediction**: Train models to forecast collision risks
2. **Activation Steering**: Apply targeted interventions to prevent unsafe actions
3. **Comprehensive Evaluation**: Measure safety and performance metrics
